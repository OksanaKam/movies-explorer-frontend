# Описание проекта

Здесь представлен cервис, в котором можно найти фильмы по запросу и сохранить их в личном кабинете.

Макет в Figma - light-2.

## Технологии

* HTML
* CSS
* JavaScript
* React
* React Router
* Разметка портирована в JSX
* Инфраструктурные файлы проекта созданы через CRA
